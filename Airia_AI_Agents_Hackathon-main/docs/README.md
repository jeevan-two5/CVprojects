# DocuSync AI — Documentation

This folder stores Markdown documentation files that are automatically updated by the DocuSync AI system.

## How it works

When a pull request is merged, the **Documentation Generation Agent** analyzes the code changes and updates (or creates) the relevant Markdown file in this folder.

## Structure

```
docs/
├── README.md          ← This file
├── api/               ← API endpoint documentation
├── modules/           ← Module-level documentation
└── changelog.md       ← Auto-generated change log
```

---

*This documentation is maintained automatically by DocuSync AI. Manual edits may be overwritten on the next PR merge.*
